Here you can find some information and the link to the "Paiement BDM" web application I am currently developing.

Information


Paiement BDM is a Web Application that replaces the existing tools I built to manage the payments of the players of my football club BDM FC.


General information

The web application is still under development, so far you can load an existing event and play with the data :
First, enter an event id from 1 to 82, it loads the corresponding event.
Then you can select the players who took part to this event and add their payment if they payed.
There are some validators (like total number of players and total payment) to make sure the data entered is correct.

Technical information

This responsive Web Application is made of 2 separates parts : an Angular Web frontend and a Java backend.
It uses Spring Boot and relies on a micro-service architecture that retrieves the data from a MySQL database.
